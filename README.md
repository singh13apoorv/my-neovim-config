# my-neovim-config
This is a changing directory of my personal neovim configuration

# How to use this neovim configuration for yourself
if you have a UNIX/LINUX machine do the following: <br>
1> first go to your home directory. <br>
2> mkdir ~/.config <br>
3> cd ~/.config <br>
4> mkdir nvim <br>
5> clone this repository in the nvim folder you just created <br> 
    git@github.com:singh13apoorv/my-neovim-config.git <br>
    --or <br>
    https://github.com/singh13apoorv/my-neovim-config.git <br>
6> nvim lua/apoorvsconfig/packer.lua <br>
7> do [Esc]:PackerSync and enter. <br>
<br>
8> go back to home directory. <br> 
9> mkdir ~/.vim <br>
10> cd ~/.vim <br>
11> mkdir undodir.<br>
### after these steps you will be able to use all the features of my configuration
